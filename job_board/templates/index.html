Job Board is a job search web application with the core functionality of aggregating recent job listings across various sites. This task is accomplished by using a web scraping combination of the requests and BeautifulSoup4 libraries.

Currently the application only scrapes from StackOverflow and Monster, but is tailored to expand to other websites.
